## 주요 기능

### Multi Step Form (Multi Step: 여러 단계 + Form: 사용자에게 입력을 받기 위한 방식 = 여러 단계로 나누어진 폼(양식))

<u>단계를 통해서 하나씩 노출을 하고, 사용자에게 자연스럽게 유도하도록 하는 것이 목표.</u>

- 협소한 화면 공간
- 많은 질문 수
- 조건부 질문
